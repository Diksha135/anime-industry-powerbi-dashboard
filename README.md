# 🎬 Global Anime Industry Analytics Dashboard (Power BI)

## 📌 Project Overview
This project presents an **Anime Industry Analytics Dashboard** built using **Microsoft Power BI**.  
The dashboard analyzes anime dataset information such as popularity, ratings, genres, studios, and media types to generate meaningful insights about trends in the anime industry.

The goal of the project is to transform raw anime data into **interactive visual analytics** that help users easily explore patterns and industry insights.

---

## 📊 Dashboard Features
The dashboard includes multiple interactive visualizations:

- **Total Anime Titles** – Number of anime entries in the dataset
- **Average Anime Rating** – Overall rating trend
- **Top Anime by Popularity** – Most popular anime titles
- **Media Type Distribution** – Distribution of TV, Movie, OVA, etc.
- **Top Production Studios** – Studios producing the most anime
- **Genre Analysis** – Popular anime genres

These visualizations allow users to quickly understand key patterns in anime production and audience preferences.
